Veck.io Keybinds Guide

A good keybind layout disappears while you play. If you still think about the key before the action, the layout is costing you time.

Last reviewed April 7, 2026Notes from the veck.app editor

Current baseline layout

The current reference in this site uses LMB for shoot, RMB and E for aim, R for reload, C and LeftShift for crouch, Tab for leaderboard, M for weapon swap, and V for inspect.

What to simplify

  • Keep reload on a key you never miss under stress.
  • Avoid stacking too many actions on neighboring keys.
  • Use duplicate binds only when they serve a real purpose.

When to change a keybind

Only change a bind if you repeatedly miss it in real matches. Do not change binds because a layout looks clever in a screenshot.
